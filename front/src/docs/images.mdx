# images in the project

All images to be used in this project should be saved inside the wp/wp-content/uploads folder.
The url to this folder is saved in the .env file under the variable: 'VITE_IMAGE_PATH'.
The current path is using localhost:8000 - used while working localy.
You can access the image folder using 'import.meta.env.VITE_IMAGE_PATH'.
The src path to a an image should be 'import.meta.env.VITE_IMAGE_PATH + "yourImage.png"'.
